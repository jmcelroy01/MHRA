Instructions:

This page converts Google Scholar MLA citations to the MHRA citation format.

Use the following input format: Last name, First name. "Title." Publisher/Journal Issue # (Year): Page Number(s).

Only a single author name works for now. The page numbers can be a single number or a range. If there are weird characters in the title, it may not convert correctly.

Report issues to me or at the GitHub Issues page.
